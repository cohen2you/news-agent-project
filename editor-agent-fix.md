# Editor Agent Trello Notes Enhancement

The editor agent has been updated to include comprehensive review notes on Trello cards. Here's what will be displayed:

## For Cards in "Submitted" List (Approved Articles):

### Editor Review Summary
- Review result (APPROVED/NEEDS REVISION)
- Revision attempt number
- Number of issues found
- List of checks performed:
  - Accuracy check
  - Completeness check
  - Prompt adherence check
  - Factual consistency check
  - Quality assessment

### Editor's Review Notes
- Detailed review notes from the AI editor
- Explanation of what was checked and results

### Revision History (if applicable)
- History of all revision attempts and feedback

## For Cards in "Needs Review" List (Escalated Articles):

### Editor Review Summary
- Same summary as above

### Editor's Detailed Review
- Full review notes explaining the issues

### Issues Found
- Numbered list of specific issues identified

### Revision History
- Complete history of all revision attempts

### Reason for Escalation
- Why the article was escalated to human review

All of this information is automatically added to the Trello card descriptions and comments when articles are reviewed.
